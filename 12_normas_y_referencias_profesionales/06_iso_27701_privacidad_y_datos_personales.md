# ISO/IEC 27701: privacidad y datos personales

## Qué te aporta
Te sirve como referencia para pensar la privacidad dentro del tratamiento de datos personales.

## Lo útil para tu contexto
- finalidad de los datos
- minimización
- acceso por necesidad
- gestión responsable de exportaciones y respaldos
- alineación entre privacidad y seguridad

## Regla práctica
No necesitas tratarla como un ritual, pero sí como una brújula cuando el sistema guarda información de personas.
