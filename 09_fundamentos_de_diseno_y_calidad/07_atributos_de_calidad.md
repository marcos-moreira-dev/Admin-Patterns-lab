# Atributos de calidad

## Propósito
Dar un vocabulario para evaluar software sin quedarse en “me gusta” o “no me gusta”.

## Atributos importantes para apps administrativas
### Adecuación funcional
Resuelve lo que el negocio realmente necesita.

### Usabilidad
Se entiende y se usa con claridad.

### Fiabilidad
No falla con facilidad en operación normal.

### Seguridad
Protege acceso, datos y operaciones críticas.

### Mantenibilidad
Se puede corregir, extender y leer sin sufrimiento excesivo.

### Testabilidad
Es posible comprobar comportamiento de forma razonable.

### Observabilidad
Cuando algo falla, se puede entender qué pasó.

### Rendimiento
Responde con tiempos aceptables para el contexto real.

### Consistencia
Las reglas, respuestas y comportamientos siguen un patrón estable.

## Cómo usar esto
Estos atributos sirven para discutir mejoras, prioridades y riesgos con un lenguaje más profesional.
