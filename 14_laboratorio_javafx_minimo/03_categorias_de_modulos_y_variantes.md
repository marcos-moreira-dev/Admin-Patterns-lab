# Categorías de módulos y variantes del laboratorio

## Categorías principales

- hub de acceso
- dashboard
- catálogo / CRUD
- wizard / flujo guiado
- expediente / caso vivo
- bandeja / cola operativa
- configuración
- aprobación / revisión
- reportes
- búsqueda / consulta especializada
- documental
- agenda / calendario

## Variantes sugeridas

### Dashboard
- dashboard simple
- dashboard con alertas
- dashboard con KPIs y tabla

### CRUD
- CRUD simple con tabla y filtros
- CRUD con detalle lateral
- CRUD con formulario modal

### Wizard
- wizard lineal
- wizard con resumen final
- wizard con ramas ligeras

### Expediente
- expediente con tabs
- expediente con acciones por estado
- expediente con historial reforzado

### Bandeja
- bandeja con preview
- bandeja con acciones masivas
- bandeja con filtros fuertes

### Reportes
- reporte tabular con filtros
- reporte con resumen y gráfico

### Búsqueda
- búsqueda avanzada con filtros
- búsqueda con panel lateral y detalle

### Documental
- lista de documentos con metadatos
- documental con preview y descarga simulada

### Agenda
- agenda básica por fecha
- agenda con lista de eventos del día

## Criterio de selección

Cada variante debe existir para mostrar un patrón útil, no para inflar el laboratorio artificialmente.
