# Frontend o desktop en entrega

## Idea principal
Entregar una interfaz no es solo mandar archivos. Debe quedar claro:
- cómo se ejecuta
- de qué backend depende
- cómo se configura
- qué versión es
- cómo se actualiza

## Para desktop
- paquete o ejecutable claro
- instrucciones de instalación o ejecución
- URL del backend configurable si aplica
- carpeta de configuración externa si conviene
- versión visible

## Para frontend web
- build compilado
- URL del backend o proxy definido
- configuración por entorno
- ruta de despliegue clara

## Qué no olvidar
- mensajes de error razonables
- manejo básico si el backend no responde
- documentación de versión
