# CI/CD fundamentos

## Significado
- CI: integración continua
- CD: entrega o despliegue continuo, según contexto

## Idea simple
Automatizar pasos repetitivos y frágiles:
- compilar
- ejecutar pruebas
- validar estilo o calidad mínima
- generar artefactos
- opcionalmente desplegar

## Qué te conviene como freelance
No necesitas pipelines gigantes desde el día uno.
Sí te conviene entender al menos:
- build automático
- test automático mínimo
- validación de que el proyecto compile
- generación de artefacto

## Beneficio real
- menos errores manuales
- más confianza en cambios
- historial claro de entregas
- base para crecer después

## Regla conservadora
Empieza pequeño:
1. compilar automáticamente
2. correr pruebas esenciales
3. generar artefacto
4. si el proyecto lo amerita, automatizar despliegue
