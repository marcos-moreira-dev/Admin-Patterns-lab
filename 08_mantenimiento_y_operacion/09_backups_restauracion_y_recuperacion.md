# Backups, restauración y recuperación

## Idea principal
El backup no sirve si nunca pensaste cómo restaurar.

## Qué mínimo conviene definir
- qué se respalda
- con qué frecuencia
- dónde se guarda
- quién puede restaurar
- cuánto tiempo se conserva
- cómo se valida que sirva

## Qué suele necesitar respaldo
- base de datos
- archivos o documentos subidos
- configuración crítica
- scripts de migración

## Preguntas útiles
- si se pierde la base hoy, qué recuperas
- cuánto tiempo de información puedes perder sin desastre
- qué harías ante borrado accidental o corrupción

## Regla práctica
Piensa en recuperación, no solo en copia.
