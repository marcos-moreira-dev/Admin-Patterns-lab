# Primera reunión: descubrimiento

## Propósito
La primera reunión no debería cerrar el sistema. Debería abrir el problema correctamente.

## Qué se busca
- contexto del negocio
- dolor principal
- urgencia real
- valor esperado
- disposición a seguir

## Qué no se busca todavía
- pantallas finales
- arquitectura técnica detallada
- precio definitivo
- lista completa de funciones
