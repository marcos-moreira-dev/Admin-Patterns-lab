# Accesibilidad en web administrativa

## Qué importa en una admin app web
- navegación por teclado
- orden de foco razonable
- labels visibles y asociados
- mensajes de error claros
- tablas comprensibles
- botones distinguibles
- no depender solo del color para estado o prioridad

## Qué revisar
- contraste de texto y fondos
- formularios con labels reales
- botones con texto claro
- modales manejables con teclado
- filtros y resultados comprensibles

## Regla práctica
Una app administrativa web no necesita verse futurista. Necesita ser entendible, utilizable y predecible.
