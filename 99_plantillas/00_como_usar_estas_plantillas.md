# Cómo usar estas plantillas

Estas plantillas no reemplazan criterio. Sirven para no empezar cada proyecto desde cero y para conservar consistencia entre levantamiento, modelado, diseño y propuesta.

## Regla práctica
- Primero piensa el caso real.
- Luego elige la plantilla que mejor te ayuda a capturarlo.
- Adapta la plantilla al dominio; no adaptes el dominio a la plantilla.

## Orden sugerido
1. `plantilla_reunion.md`
2. `plantilla_flujo.md`
3. `plantilla_entidad.md`
4. `plantilla_modelo_de_datos.md`
5. `plantilla_modulo.md`
6. `plantilla_regla_de_negocio.md`
