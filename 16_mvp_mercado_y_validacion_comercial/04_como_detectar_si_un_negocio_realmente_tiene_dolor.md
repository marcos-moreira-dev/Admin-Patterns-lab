# Cómo detectar si un negocio realmente tiene dolor

## Qué significa "dolor"

Dolor no es que el cliente diga "me gustaría una app". Dolor es cuando una fricción del negocio ya se siente como pérdida, desorden, retraso, duplicación o cansancio operativo.

## Tipos de dolor más comunes

- errores repetidos
- retrasos
- pérdida de información
- falta de control de estados
- cobros mal registrados
- inventario mal llevado
- dificultad para buscar historial
- exceso de trabajo manual
- decisiones tomadas a ciegas

## Cómo se detecta en conversación

### Señales verbales
- "eso a veces se nos pierde"
- "eso toca revisarlo a mano"
- "si no está fulano, no sabemos"
- "a veces se olvida"
- "eso lo llevamos en un cuaderno"
- "hay que preguntar varias veces"

### Señales operativas
- varios pasos fuera del sistema
- dependencias fuertes de una persona
- demasiada mensajería informal
- mucho copiar y pegar
- ausencia de estados claros
- falta de evidencia documental

## Preguntas útiles

- ¿qué parte del trabajo les genera más desorden?
- ¿qué error se repite más?
- ¿qué cosa les quita más tiempo?
- ¿qué pasa cuando alguien olvida registrar algo?
- ¿qué información les gustaría tener más clara hoy?
- ¿qué revisan a mano todos los días o todas las semanas?

## Cómo diferenciar deseo de dolor real

### Deseo
"sería bonito tener..."

### Dolor real
"esto nos está afectando porque..."

El deseo es una idea. El dolor real ya tiene costo, frecuencia y consecuencia.

## Señales de que sí hay base para una propuesta

- el cliente logra contar casos concretos
- hay ejemplos reales recientes
- el problema se repite
- el impacto es visible
- la mejora se puede traducir en orden, tiempo, control o cobro

## Señales de que todavía no

- todo es muy abstracto
- el cliente cambia de idea cada cinco minutos
- no puede contar ni un solo caso real
- el problema es difuso y poco frecuente
- lo que pide es más imagen que solución operativa

## Regla práctica

Si no puedes nombrar con una sola frase el dolor principal del negocio, todavía no estás listo para proponer el MVP.
