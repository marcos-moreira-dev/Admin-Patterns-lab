# Caso aplicado: negocio genérico

## Objetivo
Tener un molde neutro para practicar sin casarse con un dominio puntual.

## Estructura mental base
Todo negocio administrativo pequeño o mediano suele poder describirse con esta grilla:

### Catálogos
- clientes
- productos o servicios
- usuarios
- categorías
- configuraciones mínimas

### Procesos
- registrar operación principal
- dar seguimiento a algo
- cobrar o registrar movimiento
- emitir documento
- consultar estado o historial

### Flujos
- recepción o inicio
- validación
- ejecución
- cierre
- seguimiento o excepción

### Módulos
- CRUD
- wizard
- expediente
- bandeja
- dashboard
- configuración
- aprobación
- reportes
- búsqueda
- documental

## Preguntas universales
- ¿qué hacen todos los días?
- ¿qué les genera desorden?
- ¿qué se repite?
- ¿qué debe quedar registrado?
- ¿qué se debe buscar rápido?
- ¿qué cambia de estado?
- ¿qué se cobra o se entrega?
- ¿qué documentos existen?
- ¿qué cosas necesitan seguimiento?

## Uso recomendado
Este archivo sirve como base cuando todavía no tienes claro el dominio exacto o cuando quieres practicar el método sin ruido sectorial.
